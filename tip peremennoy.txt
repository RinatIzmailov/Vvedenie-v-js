                ТИПЫ ПЕРЕМЕННЫХ (ЕСТЬ 6 ПРИМИТИВНЫХ ТИПОВ)

1) string -это строка

2) boolean - всего два значения true или false

3) number - число

4) null - имеет единственное значение null. Гворит о том что сейчас значения как бы нет но оно типа есть.(позже будет нормальное значение)

6) undefine - единственное значение undefine. Говорит о том что нет значения у переменной( не определено)

7) symbol - символ (используется очень редко)


                ССЫЛОЧНЫЙ ТИП ПЕРЕМЕННОЙ OBJECT (ОН ОДИН)

ПРИМИТИВНЫЕ ПЕРЕМЕННЫЕ - непосредственно занимают определенное место в памяти. Они содержат значение в памяти ПК


ССЫЛОЧНЫЙ ТИП - переменная хранит ссылку на объект в памяти

МАССИВ (забегая вперед) - переменная которой присвоен массив тоже хранить только ссылку на ОБЪЕКТ в памяти

Можно создавать сколько угодно переменных которые БУДУТ ССЫЛАТЬСЯ НА ОДИН И ТОТ ЖЕ ОБЪЕКТ В ПАМЯТИ

        НО!!! Если поменять значение в одной переменной то это изменит сам объект в памяти и как следствие други переменные,
которые ссылаются на этот же объект тоже изменятся!

// const objectA = {
