Самое важное в JS:

1 ВЫРАЖЕНИЯ

2 ФУНКЦИИ

3 ОБЪЕКТЫ

Главная идея JS : ПРАКТИЧЕСКИ ВСЕ СУЩНОСТИ ЯВЛЯЮТСЯ ОБЪЕКТАМИ

Обект - это набор свойств "ИМЯ (КЛЮЧ) : ЗНАЧЕНИЕ"

Пример объекта:

{
    visible: true,              Слева имя оно же ключ, справа значение
    colorDepth: 24,
    title: 'My Image',
    orientation: {              В данном случае значением может быть объект (ЭТО ВЛОЖЕННЫЙ ОБЪЕКТ)
        angle: 0,
        type: 'landscape'
    },
    pixelDepth: 24,
    width: 1440
}

У console есть три метода

console.log
consol.dir    - отображает все свойства объекта
console.table - выводит в виде таблицы
